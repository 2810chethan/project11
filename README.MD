 Streamlit Graph Dashboard

An interactive and professional data visualization dashboard built using Streamlit, Plotly, Matplotlib, and Seaborn.  
Users can upload CSV or Excel files and generate multiple types of graphs dynamically.



 Features

- Upload CSV Files
- Upload Excel Files
- Interactive Dashboard
- Dark Theme UI
- Custom CSS Styling
- Responsive Layout
- Multiple Data Visualizations


 Graphs Included

- Line Chart
- Bar Chart
- Scatter Plot
- Histogram
- Box Plot
- Pie Chart
- Area Chart
- Heatmap



 Technologies Used

- Python
- Streamlit
- Pandas
- Plotly Express
- Matplotlib
- Seaborn
- OpenPyXL
